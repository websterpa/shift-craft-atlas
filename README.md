# Shift Craft (Atlas) | UK Staff Rostering SaaS

A premium UK staff shift rostering system with built-in WTR compliance and real-time labor costing.

## Documentation

Project documentation has been moved to the [docs/](./docs/) directory.

## Project Structure

- `src/`: Source code (JS, CSS)
- `tests/`: End-to-end tests (Playwright)
- `pattern-library/`: JSON definitions of shift patterns
- `docs/`: Project documentation and plans

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm start
   ```

## Testing

Run the automated test suite:
```bash
npx playwright test
```
