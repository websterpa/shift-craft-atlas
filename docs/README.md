## üìñ About Shift Craft (Atlas)

Shift Craft (Atlas) is a shift rostering application designed to help managers create compliant, efficient work schedules while adhering to UK employment law.

**Key Features**:
- Shift assignment with clash detection
- UK WTR 1998 compliance monitoring
- Pattern library (Continental, DuPont, NHS, etc.)
- AI-powered pattern recognition
- Payroll preview and export
- Attendance tracking
